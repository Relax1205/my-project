# Мой первый проект

## Проект  
Проект по дисциплине «Основы фронтенд-разработки».

### Реализованы требования ПЗ №5 (мобильный интерфейс):
- **Гамбургер-меню** с оверлеем на мобильных устройствах (≤ 768px)
- **Фиксированная шапка** (`position: sticky`) на мобильных
- **Кнопка «Наверх»** с плавной прокруткой, появляющаяся после прокрутки >300px
- **Размеры touch-зон ≥ 44×44 px** (все интерактивные элементы соответствуют рекомендациям HIG и Material Design)
- **Шрифты**:
  - ≥ 16px — Material Design
  - ≥ 17px — Human Interface Guidelines (на экранах ≤ 480px)
- **Обратная связь** при наведении и нажатии (анимации, hover-эффекты, focus-состояния)
- **Отсутствие горизонтального скролла** на всех мобильных разрешениях
- **Валидный HTML/CSS/JS** (соответствует стандартам и WCAG)
- Отдельные файлы:
  - `mobile.css` — стили для мобильных устройств
  - `menu.js` — логика гамбургер-меню и кнопки «Наверх»

## Используемые технологии  
- HTML5  
- CSS3  
  - Flexbox  
  - CSS Grid  
  - Masonry-раскладка  
  - Медиазапросы  
- Семантическая вёрстка  
- Методология БЭМ  
- JavaScript (для интерактивности меню и скролла)  
- GitHub Pages
