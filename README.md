# Мой первый проект

## Проект  
Проект по дисциплине «Основы фронтенд-разработки»

Реализованы следующие требования ПЗ №4:
- **Семантическая разметка** с использованием HTML5 (`<header>`, `<main>`, `<section>`, `<footer>` и др.)
- **Адаптивное навигационное меню** с помощью **Flexbox**: на десктопе — горизонтальное, на мобильных — переключается в вертикальную ориентацию
- **Двумерные сетки контентных блоков** реализованы с использованием **CSS Grid**
- **Masonry-раскладка** в разделе «Inspiration Gallery»: элементы разной высоты корректно располагаются в автоматической сетке
- **Адаптивность по принципу Desktop First** с медиазапросами и следующими **breakpoints**:
  - `1199px` (ноутбуки)
  - `991px` (планшеты)
  - `767px` (мобильные, альбомная ориентация)
  - `575px` (мобильные, портретная ориентация)
  - `399px` (очень маленькие экраны)
- Именование классов выполнено в соответствии с методологией **БЭМ**
- Отсутствует горизонтальный скролл на мобильных устройствах; изображения и текст не выходят за границы контейнеров

## Используемые технологии  
- HTML5  
- CSS3  
  - Flexbox  
  - CSS Grid  
  - Masonry-раскладка  
  - Медиазапросы
- Семантическая вёрстка  
- Методология БЭМ  
- GitHub Pages
